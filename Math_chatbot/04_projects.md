# Projects and Practical Applications

Our philosophy is "Learn by Doing." Throughout the cohort, you will work on practical projects that apply mathematical theories to real-world scenarios.

## 1. The PageRank Algorithm Model (Linear Algebra)
Construct a simplified version of Google's foundational PageRank algorithm using matrix operations and eigenvectors to rank a mini-web of interconnected pages.

## 2. Optimization Engine (Calculus)
Build a mathematical model to optimize the supply chain logistics for a hypothetical e-commerce company, utilizing multivariable calculus and gradient descent techniques.

## 3. Election Prediction Model (Probability & Statistics)
Analyze historical polling data to build a probabilistic model that predicts election outcomes, applying concepts of statistical distributions and confidence intervals.

## 4. Capstone Project
A final, open-ended project where you select a real-world dataset or problem, formulate a mathematical model, solve it, and present your findings to the cohort.
