# Complete Curriculum

## Module 1: Foundations of Logic and Proofs (Weeks 1-2)
- Propositional logic and boolean algebra
- Direct and indirect proofs, induction
- Set theory fundamentals

## Module 2: Linear Algebra for the Modern World (Weeks 3-5)
- Vectors, matrices, and linear transformations
- Eigenvalues, eigenvectors, and diagonalization
- Applications in computer graphics and machine learning

## Module 3: Calculus and Optimization (Weeks 6-8)
- Limits, continuity, and derivatives
- Integration and the Fundamental Theorem of Calculus
- Multivariable calculus and gradient descent

## Module 4: Probability and Statistics (Weeks 9-11)
- Combinatorics and discrete probability
- Continuous distributions (Normal, Poisson, Exponential)
- Hypothesis testing and statistical inference

## Module 5: Capstone and Applications (Week 12)
- Bringing it all together: real-world math modeling
- Final project presentations
